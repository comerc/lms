Этот навык предоставляет полный контекст о проекте, над которым ведется разработка.

## Контекст проекта

**Тип проекта:** EdTech платформа для подготовки к ЕГЭ

**Технологический стек:**

- **Фронтенд:** Bun + React + TypeScript + Tailwind CSS + Ant Design
- **Бэкенд:** Go (Chi router, sqlx для БД)
- **AI:** Claude API

**Геймификация:**

- XP (очки опыта)
- Уровни
- Стрики (последовательные дни занятий)
- Лидерборды

**Целевая аудитория:** Школьники 9-11 класс

## Примеры использования

### Ввод пользователя:

"Создай компонент для отображения прогресса"

### Ожидаемый вывод:

Компонент должен учитывать систему XP, уровней и стриков, быть понятным для школьников 9-11 класса, использовать Ant Design и Tailwind CSS.
